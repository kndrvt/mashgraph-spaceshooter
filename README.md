### **Компиляция и запуск**  

#### **Библиотеки**   

###### **SOIL**
-  sudo apt-get install libsoil-dev  

###### **GLM**
- git clone https://github.com/g-truc/glm.git
- cd glm
- cmake ./
- make
- sudo make install

###### **ASSIMP**  
- git clone https://github.com/assimp/assimp.git
- cd assimp
- cmake CMakeLists.txt
- make -j4
- sudo cp -a include /usr/local
- sudo apt-get install libassimp-dev

#### **Сборка**
- mkdir build
- cd build
- cmake ..
- make

#### **Запуск**
- ./main

### **Базовая часть**   

- Реализованы 3 различных вида врагов. Каждый враг получает соответствующий виду урон и умирает за 1, 2 или 3 попадания. 
За попадание даётся 10 очков, таким образом, за убийство игрок получает 10, 20 или 30 очков.
    - Враги стреляют в игрока раз в 5 секунд. Вражеские пули летят на расстояние не более 30.
    - При попадании снаряда враг подсвечивается красным. При гибели разлетаются на меши.  
    - При выстреле вылетает объект-пуля. Ранение происходит при попадании центра пули в сферообразную зону поражения. 
    (Если это подходит под "более сложные методы", то **(+1)**)  
    
- Реализованы 2 вида астероидов. Астероиды разрушаются после 1 попадания. За каждый астероид даётся 10 очков.
    - При попадании астероиды разлетаются на меши.   
    
- Космическое пространство реализовано с помощью скайбокса и мелких частиц, летящих на игрока. 
 
- Реализован прицел, находящийся в центре экрана. 
При движении мыши меняется обзор камеры, что обеспечивает смещение прицела.  

- Выстрел происходит при нажатии левой кнопки мыши (также правой, как требуется в задании). 
Все пули визуализированы. Пули летят в направлении прицела.  

- Сверху слева отображается количество здоровья, сверху справа - количество очков.  

- Столкновение корабля игрока с кораблём противника или астероидом эквивалентно попаданию пулей. 
Также предусмотрено столкновение корбаля противника с астероидом. 
В обоих случаях корабль противника и/или астероид разрушаются.  

- Если здоровье достигает 0, игра незамедлительно завершается.

### **Другие фичи**

- **"Режим Бога"** (*клавиша 9*)  
Необходим скорее для отладки. В нём здоровье не уменьшается. Появляется возможность свободно двигаться по сцене. 
Враги продолжают двигаться в том же направлении, но стреляют по-прежнему в игрока.  

- **Дефолтное положение** (*клавиша 0*)  
Камера возвращается в исходное положение, "режим Бога" отключается, но здоровье и очки остаются прежними.

- **Изменение угла обзора**  
При прокрутке колёсика мышки угол обзора может меняться от 30 до 45 градусов. По умолчанию 45 градусов.
